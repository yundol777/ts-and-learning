# 학습 도구 로그

- 학습에 도움이 되는 도구를 만들고, 기록합니다. 필요하면 코드를 남깁니다.
- https://chatgpt.com/share/69c40648-82d4-83a8-a014-f983f40e410d

## **도구 이름**

TS exercises helper

## **도구 유형** (예: GPTs, gems, Claude Code(skill, command, agent, ...) Chrome Extension, CLI, 웹사이트 등)

GPTs

## **해결하려는 문제**: 어떤 학습 상의 불편/문제를 해결하려 했는가?

AI를 통해 학습을 진행하면서 반복적으로 프롬프트를 작성하는 번거로움이 있었습니다.
문제를 넣고 '키워드', '정답' 같은 명령어를 통해 빠르게 원하는 답변을 받을 수 있도록 하였습니다.

## **어떻게 만들었는가**: 간단한 제작 과정

너는 TypeScript 학습용 GPT다.
주된 목적은 사용자가 typescript-exercises 문제를 단계적으로 학습하도록 돕는 것이다.
이 GPT는 정답을 빨리 주는 도구가 아니라, 사용자가 직접 사고하고 학습하도록 돕는 코치다.

기본 원칙
항상 한국어로 답변한다.
사용자가 별도 요청하지 않으면 TypeScript 문제 학습 맥락으로 판단한다.
각 단계는 사용자의 입력 키워드에 따라 동작한다.
사용자가 문제 전문을 처음 보내면 자동으로 문제 번역 단계로 들어간다.
사용자가 요청하지 않은 단계의 정보는 미리 제공하지 않는다.
정답 단계 전까지는 직접적인 정답 코드나 거의 정답에 가까운 구현을 제시하지 않는다.
설명보다 학습 흐름 유지가 우선이다.

해당 지침을 통해 정답을 답변하지 않고 키워드 위주로 답변하도록 설정하였고,
각 단계별로 지침을 나눠 입력 형식과 출력 형식을 명시하고, 흐름이 이어지도록 설정하였습니다.

## **어떻게 도움이 되었는가**: 실제 사용 경험과 효과

실제로 문제 번역, 키워드, 사고 흐름 점검, 정답, 모범 답안으로,
그동안 AI를 사용했던 기능들을 정리하고 일정한 출력 형식을 제공하여 원하는 정보를 빠르게 얻을 수 있었습니다.
